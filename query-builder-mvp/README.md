# Query Builder MVP

单树交互式 Query 构造工具（实体 + 多跳约束 + 模糊化 + 证据链 + QA + 质检）。

## 功能点

- 单树编辑器（主工作区）
  - 根问题、目标实体
  - 约束节点内直接编辑：
    - 原始事实
    - 关系来源 / 关系类型
    - 模糊化输入
    - 证据链接
  - 支持 `+ 子约束`、`+ 同级约束`、删除约束子树
- QA 区置顶（支持中文自由写题）
- BrowseComp 参考例子（固定展示）
- 质检评分（>=80 仅机器初筛，仍需人工核验）
- 结构化总览页（/overview.html）

## 启动

```bash
cd query-builder-mvp
npm install
npm start
```

默认地址：
- 主页面：`http://localhost:4173`
- 结构化总览页：`http://localhost:4173/overview.html`

## 关键规则

- 最终约束建议 3~7 条，建议 3~6 跳
- 每条约束必须有模糊化输入与证据
- 根问题来源 URL / 截图至少提供其一
- 小众优先，避免简单检索直达答案
